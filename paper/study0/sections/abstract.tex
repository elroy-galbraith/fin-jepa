\begin{abstract}
We evaluate the FT-Transformer architecture as a standalone encoder for structured financial data extracted from SEC EDGAR XBRL filings, benchmarking it against logistic regression, XGBoost, and gradient-boosted trees on four financial distress prediction tasks. Using a strictly temporal train/test split on 33{,}737 company-year observations (2012--2023), we find that the tuned FT-Transformer is directionally positive on all four outcomes against default-hyperparameter XGBoost (+0.015 to +0.075 AUROC), with statistically significant gains on two (paired bootstrap, $p < 0.05$). Against Optuna-tuned baselines (30 trials per model), the strict AUROC gate is not met (2/4 wins); however, the FT-Transformer retains calibration advantages on the highest-prevalence outcome and produces more informative probability distributions on rare-event outcomes. Self-supervised pretraining via masked feature reconstruction provides additional lift, particularly on low-base-rate outcomes (+0.092 AUROC on SEC enforcement actions at mask ratio 0.50). Multi-seed evaluation (3 seeds) confirms that FT-Transformer variance is modest (std $\leq 0.013$). We argue that calibration quality and representation-learning potential---evidenced by SSL pretraining gains---justify proceeding with the FT-Transformer as the base encoder for a larger research program investigating cross-modal joint-embedding predictive architectures for financial risk signals. All data extraction pipelines, feature definitions, and training code are released as open-source software.
\end{abstract}
